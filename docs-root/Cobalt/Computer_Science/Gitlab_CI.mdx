# gitlab-ci.yml

The **gitlab-ci.yml** file is responsible for defining how the CI pipeline runs. For details about the syntax of this file, see the [Gitlab Wiki page](https://docs.gitlab.com/ce/ci/yaml/).



# Terminology

Job - A single section of CI executed on a runner.

Pipeline - A sequence of jobs in parallel or sequence. See the relevant [Wiki page](https://docs.gitlab.com/ce/ci/pipelines.html#introduction).

Runner - A machine that performs the actual execution of a job.
