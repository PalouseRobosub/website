==== VPN ====

=== On Robosub Server ===

1. sudo su (must be in sudoers file)

2. cd /etc/openvpn/easy-rsa/

3. source vars

4. ./build-key <name>

5. Use all defaults, y to confirm and commit

6. cd keys

7. tar -cf <name>.tar ca.crt client.conf <name>.crt <name>.key

8. chown root:robosub_admin <name>.tar

9. mv <name>.tar /home/robosub/vpn_keys/

=== On Client Machine ===

10. install openvpn on client

11. cd /etc/openvpn/

12. rsync -Ph <user>@robosub.eecs.wsu.edu ./

13. tar -xf <name>.tar

13. modify client.conf to point at files

  Under SSL/TLS parms.
  ca ca.crt
  cert <name>.crt
  key <name>.key

14. sudo service openvpn@client restart
