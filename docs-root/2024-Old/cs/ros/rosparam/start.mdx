====== Using ROS Param ======

This command allows for retrieval and manipulation of the parameter server.

==== Get Params ====

  rosparam get <PARAM>
  
==== Set Params ====

  rosparam set <PARAM> <VALUE>
  
==== Load Params ====

  rosparam load <FILE>
  
==== Dump Params ====

  rosparam dump <FILE>
  
==== List Params ====

  rosparam list
